# Tooltip

# Default

Documentation Vuesax Alpha

These docs refer to the Vue 3 support, to see the documents of the previous versions you can do it here ðŸ‘‰ Vuesax (4.0+)

Easily add a tooltip with the vs-tooltip component, the content within the default slot is the parent element of the tooltip and the one that triggers to show or hide it

The tooltip slot is the content inside the tooltip

Code Copied
  • Template 
<template>
  <div class="center">
    <vs-tooltip>
      <vs-button type="flat"> Do hover here </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
  </div>
</template>

# Placement

Change the placement of the tooltip with the properties

  • top
  • bottom Default
  • left
  • right
Code Copied
  • Template 
<template>
  <div class="center">
    <vs-tooltip placement="left">
      <vs-button type="border"> left </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip placement="top">
      <vs-button type="border"> Top </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>

    <vs-tooltip placement="right">
      <vs-button type="border"> right </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
  </div>
</template>

# Color

Use the color property to change the base color of the component and some of the child components, to better understand the handling of colors and themes you can see it here

Allowed values are:

  • primary
  • success
  • danger
  • warning
  • dark
  • RGB
  • HEX
Code Copied
  • Template 
<template>
  <div class="center">
    <vs-tooltip>
      <vs-button type="shadow"> Default </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip color="primary">
      <vs-button type="flat"> Primary </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip color="success">
      <vs-button color="success" type="flat"> Success </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip color="danger">
      <vs-button color="danger" type="flat"> Danger </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip color="warn">
      <vs-button color="warn" type="flat"> Warning </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip color="dark">
      <vs-button color="dark" type="flat"> Dark </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip color="#7d33ff">
      <vs-button color="#7d33ff" type="flat"> HEX </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip color="rgb(59,222,200)">
      <vs-button color="rgb(59,222,200)" type="flat"> RGB </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
  </div>
</template>

# Border

Change the style of the tooltip by adding a border and changing the color of the background with the border or border-thick property

Now the color property would change the color of the border

Code Copied
  • Template 
<template>
  <div class="center">
    <vs-tooltip type="border">
      <vs-button type="transparent"> Do hover here </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip color="danger" type="border">
      <vs-button color="danger" type="transparent"> Do hover here </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip color="#7d33ff" type="border-thick">
      <vs-button color="#7d33ff" type="transparent"> Do hover here </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
  </div>
</template>

# Not Arrow

In some cases you may need to remove the arrow from the tooltip for this you can use the not-arrow property

Code Copied
  • Template 
<template>
  <div class="center">
    <vs-tooltip :show-arrow="false" placement="left">
      <vs-button type="border"> left not-arrow </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip :show-arrow="false">
      <vs-button type="border"> Top not-arrow </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip :show-arrow="false" placement="bottom">
      <vs-button type="border"> Bottom not-arrow </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
    <vs-tooltip :show-arrow="false" placement="right">
      <vs-button type="border"> right not-arrow </vs-button>
      <template #content> This is a beautiful button </template>
    </vs-tooltip>
  </div>
</template>

# Content

We create the tooltip component in such a way that you have a lot of freedom to add anything and create all kinds of interface inside it.

Code Copied
  • Template
  • Script
  • Style
  • All 
<template>
  <div class="center">
    <vs-tooltip>
      <vs-button type="gradient"> Whats is Vuesax? </vs-button>
      <template #content>
        <div class="content-tooltip">
          <h4>Whats is Vuesax?</h4>
          <p>Vuesax is a framework of UI components created with Vuejs</p>
        </div>
      </template>
    </vs-tooltip>
    <vs-tooltip
      v-model="activeTooltip1"
      placement="bottom"
      type="shadow"
      trigger="click"
    >
      <vs-button color="danger" @click="activeTooltip1 = !activeTooltip1">
        Click Delete User
      </vs-button>
      <template #content>
        <div class="content-tooltip">
          <h4 class="center">Confirm</h4>
          <p>
            You are sure to delete this user, by doing so you cannot recover the
            data
          </p>
          <footer>
            <vs-button color="danger" block @click="activeTooltip1 = false">
              Delete
            </vs-button>
            <vs-button
              type="transparent"
              color="dark"
              block
              @click="activeTooltip1 = false"
            >
              Cancel
            </vs-button>
          </footer>
        </div>
      </template>
    </vs-tooltip>
    <vs-tooltip type="shadow" interactivity>
      <vs-avatar>
        <img src="/avatars/avatar-5.png" alt="" />
      </vs-avatar>
      <template #content>
        <div class="content-tooltip">
          <div class="body">
            <div class="text">
              Closed Tasks
              <span> 89 </span>
            </div>
            <vs-avatar
              shape="circle"
              size="80"
              @click="activeTooltip1 = !activeTooltip1"
            >
              <img src="/avatars/avatar-5.png" alt="" />
            </vs-avatar>
            <div class="text">
              Open Tasks
              <span> 8 </span>
            </div>
          </div>
          <footer>
            <vs-button shape="circle" icon type="border">
              <i class="bx bxs-share-alt" />
            </vs-button>
            <vs-button shape="circle"> Message </vs-button>
            <vs-button shape="circle" icon type="border">
              <i class="bx bx-like" />
            </vs-button>
          </footer>
        </div>
      </template>
    </vs-tooltip>
  </div>
</template>
