# Scrollbar

# Default

Documentation Vuesax Alpha

These docs refer to the Vue 3 support, to see the documents of the previous versions you can do it here 👉 Vuesax (4.0+)

Used to replace the browser’s native scrollbar.

Use height property to set the height of the scrollbar, or if not set, it adapts according to the parent container height.

# Max height

The scrollbar is displayed only when the element height exceeds the max height.

    <vs-button type="flat" @click="onDelete">Delete Item</vs-button>

  <vs-scrollbar max-height="400px">
    <p v-for="item in count" :key="item" class="scrollbar-demo-item">
