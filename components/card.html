Documentation Vuesax Alpha

These docs refer to the Vue 3 support, to see the documents of the previous versions you can do it here 👉 Vuesax (4.0+)

# Default

Add a card with the vs-card component for the structure of this component we have several slots

  • title
  • text
  • buttons
  • interactions
  • img

        <template #title>
          <h3>Pot with a plant</h3>
        </template>

          <img :src="`/foto${card}.png`" alt="" />

# API

PROPS

PropertyTypeValuesDescriptiondefaultExample More
typeString12345Change the style of the letter and the position of the internals. Usage

SLOTS

PropertyTypeValuesDescriptiondefaultExample More
textslotundefinedAdd text to the card in the section determined by the card type. Usage
titleslotundefinedAdd the title to the card in the section determined by the type of card. Usage
buttonsslotundefinedAdd the buttons to the card in the section determined by the type of card. Usage
interactionsslotundefinedAdd the interactions to the card in the section determined by the card type. Usage
imgslotundefinedAdd the image or video to the card in the section determined by the type of card. Usage
